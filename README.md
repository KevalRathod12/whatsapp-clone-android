## 📖 Project Description

This project is a **simple WhatsApp Clone** Android application developed as a **mini project for college submission**.

The application demonstrates core Android development concepts including:
- **RecyclerView** with custom Adapters
- **Firebase Authentication** (Email & Password)
- **Firebase Realtime Database** for live messaging
- **Fragment + ViewPager2** for multi-tab UI
- Clean **Material Design** UI matching WhatsApp's style

## ⚙️ Setup & Installation

### Prerequisites
- Android Studio (latest version)
- Java JDK 8+
- Firebase account (free)

### Steps

**1. Clone the repository**
```bash
git clone https://github.com/yourusername/WhatsappClone.git
cd WhatsappClone
```

**2. Firebase Setup**
1. Go to [firebase.google.com](https://console.firebase.google.com)
2. Create a new project → `WhatsAppClone`
3. Add Android app → Package: `com.example.whatsappclone`
4. Download `google-services.json` → place inside `app/` folder
5. Enable **Authentication → Email/Password**
6. Create **Realtime Database → Test Mode**

**3. Open in Android Studio**
- Open the project folder in Android Studio
- Click **Sync Now** when prompted
- Click **Run ▶️** to build and install on device/emulator

## 📱 How to Use

1. **Register** — Create an account with name, email, password
2. **Login** — Sign in with your credentials
3. **Chats Tab** — See all registered users → tap to open chat
4. **Send Messages** — Type and send real-time text messages
5. **Status Tab** — View contact status updates
6. **Calls Tab** — View call history log
7. **Logout** — Tap ⋮ (3-dot menu) → Logout
